### Please clone Demo branch



### How to run Frontend
`Yarn install`

`Yarn start`



### How to run Server
`cd server`

`python manage.py makemigrations meetingoverflow`

`python manage.py migrate`

`pip install -r requirements.txt`

`python manage.py runserver`
